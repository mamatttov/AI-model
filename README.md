# AI Model (English)

A web application for audio transcription and translation using modern AI models. Built with React and Vite, it allows users to record or upload audio, transcribe speech to text, and translate the result into multiple languages.

## Features

- Record audio directly in the browser or upload audio files
- Transcribe speech to text using advanced AI models (Whisper)
- Translate transcribed text into many languages
- Simple, modern, and responsive UI

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- @xenova/transformers (for AI models)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser at [http://localhost:5173](http://localhost:5173)

## Scripts

- `npm run dev` — start development server
- `npm run build` — build for production
- `npm run preview` — preview production build
- `npm run lint` — run ESLint

## Folder Structure

- `src/components/` — React components (UI, logic)
- `src/utils/` — utility scripts and web workers
- `public/` — static assets

# AI Model (Русский)

Веб-приложение для транскрипции и перевода аудио с помощью современных AI-моделей. Построено на React и Vite, позволяет записывать или загружать аудио, преобразовывать речь в текст и переводить результат на разные языки.

## Возможности

- Запись аудио прямо в браузере или загрузка файлов
- Транскрипция речи в текст с помощью AI-моделей (Whisper)
- Перевод транскрибированного текста на множество языков
- Простой, современный и адаптивный интерфейс

## Используемые технологии

- React 19
- Vite
- Tailwind CSS
- @xenova/transformers (AI-модели)

## Быстрый старт

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
4. Откройте браузер по адресу [http://localhost:5173](http://localhost:5173)

## Скрипты

- `npm run dev` — запуск сервера разработки
- `npm run build` — сборка для продакшена
- `npm run preview` — предпросмотр сборки
- `npm run lint` — проверка кода ESLint

## Структура проекта
